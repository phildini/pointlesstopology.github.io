Living room LED lights

You can set the light color by messaging the bot LEDzpln (thanks to marktraceur for the name :) on Freenode. It accepts messages in #ptopology:

LEDzpln: red

or privately:

/msg LEDzpln red

You can specify any CSS color value or one of the special values off or rainbow.

Although the IRC bot mostly does solid colors, the LEDs themselves are controllable in 198 individual groups of three. If you are on the Pointless internal network, you can set the colors by sending a UDP datagram with 198 little-endian 16-bit numbers of color format 0bbbbbrrrrrggggg to hostname inky (10.5.0.6), port 2772. You can use this for animations and other custom hacks. There is example code for Python.

The LEDs are powered through a labeled power strip so if you want to force the lights off (e.g. because somebody is sleeping in the living room), just turn off that power strip.
